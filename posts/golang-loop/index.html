Go言語でループ

今でもたまにやらかすのでメモっておく。

①

users := []string{"take", "hoge", "moge"}
for i := range users {
  fmt.Println(users[i]) // i is index
}

②

users := []string{"take", "hoge", "moge"}
for i, user := range users {
  fmt.Println(user) // user is value copy
}

③

users := []string{"take", "hoge", "moge"}
for _, user := range users {
  fmt.Println(user) // user is value copy
}

①が一番早い。 でも②③の方が読みやすい。
